# Deploy GitHub Pages

This repository hosts a simple HTML page for Plash MCP Integration.

The GitHub Pages branch is set to serve content from the root of the `main` branch.

To enable GitHub Pages:
1. Go to the repository Settings.
2. Scroll to the GitHub Pages section.
3. Select `main` branch as the source.
4. Save the settings.

Visit the deployed page at: https://Dragonglass101.github.io/PlashMCP-integration/
